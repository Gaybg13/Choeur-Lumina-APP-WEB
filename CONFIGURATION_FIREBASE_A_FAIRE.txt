CHŒUR LUMINA PWA — CONFIGURATION À FAIRE

1. Dans Firebase, ajouter une application Web au projet existant chorale-lumina-app.
2. Copier les valeurs firebaseConfig dans public/firebase-config.js.
3. Dans Paramètres du projet > Cloud Messaging > Web Push certificates :
   générer une paire de clés et copier la clé publique dans
   self.__FIREBASE_VAPID_KEY__ dans public/firebase-config.js.
4. Construire : npm install puis npm run build.
5. Déployer le dossier dist avec Firebase Hosting.
6. Sur iPhone : ouvrir l'URL dans Safari > Partager > Ajouter à l'écran d'accueil.
7. Ouvrir la PWA installée > Profil > Activer les notifications.

IMPORTANT :
Le projet est déjà configuré pour utiliser les mêmes collections Firebase que l'application Android.
Les notifications Web utilisent une sous-collection :
members/{memberId}/notificationTokens/{tokenDoc}
Les Cloud Functions Android actuelles devront être adaptées pour lire aussi ces tokens Web.
